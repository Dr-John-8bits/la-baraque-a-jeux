# Licence du projet Lille-Mêle

Ce dépôt utilise un régime de licence double.

## 1. Code source technique

Le code source technique du projet est distribué sous licence MIT, sauf mention contraire dans un fichier donné.

Cela concerne notamment :

- les composants applicatifs génériques ;
- la logique de jeu ;
- les hooks, helpers, scripts, tests et outils de validation ;
- la configuration technique du projet.

Le texte complet de cette licence se trouve dans `LICENSE-CODE.md`.

## 2. Contenus, grilles et identité

Les contenus éditoriaux et les éléments distinctifs de Lille-Mêle ne sont pas distribués sous licence open-source.

Ils sont protégés par :

```text

```

Cela concerne notamment :

- le nom Lille-Mêle ;
- l'identité du jeu ;
- les textes, microcopies, anecdotes et descriptions ;
- les grilles, catégories, réponses, questions et fichiers de données éditoriales ;
- les assets originaux, visuels, illustrations, sons, logos et éléments graphiques ;
- la sélection, l'organisation et l'enrichissement éditorial des contenus.

Le texte complet de cette protection se trouve dans `LICENSE-CONTENT.md`.

Les bibliothèques, dépendances, données publiques, contenus tiers et ressources externes éventuellement utilisés par le projet restent soumis à leurs licences et conditions respectives.

La licence MIT du code ne donne aucun droit sur les contenus éditoriaux, les grilles, le nom Lille-Mêle ou l'identité du jeu.

La protection des contenus Lille-Mêle ne revendique aucun droit sur les données publiques brutes, mais couvre leur sélection, leur organisation, leur adaptation, leur rédaction, leur présentation et leur enrichissement éditorial lorsqu'ils sont originaux au projet.

Toute réutilisation de données publiques doit conserver les attributions, mentions de source, dates de mise à jour et conditions imposées par les licences d'origine.
