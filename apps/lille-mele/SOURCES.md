# Sources et crédits

Ce fichier recense les sources utilisées ou identifiées pour construire les contenus de Lille-Mêle.

Les contenus intégrés au jeu doivent être rédigés originalement. Les sources servent à vérifier des faits, pas à copier des textes.

## Principes

- Attribuer les données publiques utilisées.
- Noter la date de consultation.
- Noter la licence quand elle est connue.
- Ne pas copier les descriptions éditoriales de sites tiers.
- Ne pas réutiliser de logos, plans, photos, pictogrammes ou visuels officiels sans autorisation.
- Éviter les données à licence contraignante si elles ne sont pas nécessaires au MVP.

## À faire avant lancement public

- Créer une fiche source par grille.
- Ajouter la source exacte pour chaque anecdote.
- Distinguer les faits factuels, les formulations originales et les inspirations.
- Vérifier les licences des jeux de données réellement importés.
- Ajouter une page `Sources` dans l'app si les contenus dépassent le prototype.
