# Confidentialité

Version : `26.05.31.5`

## Données collectées

Le prototype Lille-Mêle ne collecte aucune donnée personnelle côté serveur.

Il n'y a actuellement :

- aucun compte utilisateur ;
- aucun formulaire ;
- aucun backend ;
- aucun suivi publicitaire ;
- aucun outil d'analytics intégré ;
- aucune géolocalisation.

## Données stockées localement

Le jeu utilise `localStorage` dans le navigateur pour conserver :

- l'état de la partie du jour ;
- les groupes trouvés ;
- le nombre d'erreurs ;
- le streak local ;
- quelques statistiques locales simples ;
- l'état du bonus quotidien.

Ces données restent sur l'appareil de l'utilisateur.

## Suppression des données

L'utilisateur peut supprimer ces données en vidant les données de site du navigateur.

Une future version pourra proposer un bouton intégré de réinitialisation.

## Partage

Le bouton de partage génère un texte sans spoiler. Le joueur choisit ensuite lui-même où le partager.

## Évolutions futures

Si une mesure d'audience, un backend ou un formulaire est ajouté plus tard, cette politique devra être mise à jour avant publication.
