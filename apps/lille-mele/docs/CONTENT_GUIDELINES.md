# Conventions éditoriales

## Voix

La voix Lille-Mêle doit être :

- locale ;
- claire ;
- complice ;
- légèrement malicieuse ;
- jamais méprisante ;
- jamais folklorique par défaut.

## Règle absolue

Aucune référence religieuse dans :

- les cartes ;
- les titres de groupes ;
- les anecdotes ;
- les questions ;
- les réponses ;
- les microcopies ;
- les sources visibles dans le jeu si elles ne sont pas nécessaires.

## Types de contenus recommandés

- Métro et tramway.
- Quartiers de Lille.
- Communes de la MEL.
- Lieux culturels.
- Patrimoine civil et urbain.
- Architecture.
- Gastronomie locale.
- Braderie.
- Mots régionaux.
- Histoire urbaine.
- Géographie locale.
- Universités, innovation, santé, culture, sport.

## Équilibre d'une grille

Une grille quotidienne devrait idéalement contenir :

- 1 groupe facile ;
- 2 groupes moyens ;
- 1 groupe difficile ou piégeux.

## Qualité d'une catégorie

Une bonne catégorie doit être :

- vérifiable ;
- claire une fois révélée ;
- spécifique ;
- non arbitraire ;
- compatible avec exactement 4 items ;
- intéressante ou amusante.

## Pièges

Accepté :

- une association évidente mais fausse ;
- une logique plus précise que prévu ;
- un double sens contrôlé.

Interdit :

- deux solutions également valides ;
- une catégorie impossible à comprendre après révélation ;
- un item qui appartient légitimement à deux groupes de la même grille.

## Sources

Chaque grille destinée au public doit avoir une fiche de vérification :

- date de la grille ;
- catégories ;
- items ;
- source de chaque fait ;
- note sur les ambiguïtés ;
- validation éditoriale ;
- validation absence de référence religieuse.

## Microcopy

Exemples de ton acceptable :

- `Bonne correspondance.`
- `Trois sur quatre. Une carte fraude dans la rame.`
- `Grille démêlée. Tu peux descendre à la prochaine.`
- `Une nouvelle grille demain. Même quai, autre piège.`

À éviter :

- blagues répétées sur les mêmes clichés ;
- ton institutionnel ;
- références privées incompréhensibles ;
- formulations humiliantes après une erreur.
