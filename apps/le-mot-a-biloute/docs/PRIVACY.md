# Confidentialité

## Données collectées

La version actuelle ne collecte aucune donnée personnelle.

## Stockage local

Le jeu utilise `localStorage` pour conserver :

- la progression du jour ;
- les essais ;
- les indices utilisés ;
- les statistiques locales ;
- la série quotidienne.

Ces données restent dans le navigateur du joueur.

## Réseau

La version actuelle ne transmet pas les résultats à un serveur.

Le partage est déclenché par le joueur via les capacités du navigateur.

## Suppression des données

Le joueur peut supprimer les données en vidant le stockage local du site dans son navigateur.
