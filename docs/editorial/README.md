# Editorial

Ce dossier regroupera les conventions editoriales communes aux jeux :

- sources documentaires ;
- regles de reformulation ;
- exclusions et contenus sensibles ;
- schemas de corpus ;
- suivi des validations.

Les demandes editoriales propres a chaque jeu restent pour le moment dans leurs dossiers applicatifs. Elles seront fusionnees ici quand le schema commun sera stabilise.
