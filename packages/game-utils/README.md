# Game utils

Helpers partages par les jeux et le blog.

## Modules

- `daily.js` : date du jour, index quotidien, selection d'un contenu journalier.
- `fetch-json.js` : chargement JSON statique.
- `markdown.js` : rendu Markdown simple du blog.
- `random.js` : hash et melange deterministe.
- `share.js` : partage Web Share API avec fallback presse-papiers.
- `storage.js` : lecture/ecriture JSON dans `localStorage`.
- `text-render.js` : echappement HTML.

Ces modules restent volontairement petits afin de ne pas imposer un moteur de jeu unique.
